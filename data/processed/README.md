# Processed data

Final, analysis-ready datasets. Outputs from the cleaning and feature engineering steps.

## Expected contents

- `attributes_clean.csv` — cleaned wide-format longlist
- `attributes_long.csv` — melted long-format (one row per attribute × battery type)
- `attributes_enriched.csv` — adds complexity score and complexity band
- `manufacturer_disclosures_validated.csv` — validated Act 4 scores
- `battery_passport.duckdb` — the DuckDB warehouse with all 8 tables (gitignored, regenerated by `dbt build`)

## Reproducing

```bash
python src/data/clean_longlist.py
cd dbt && dbt build && cd ..
```
